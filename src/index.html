Find Good Films

Find Good Films is a movie database where you can create your favourite list of movies. You can search movies or browse between Popular, New Releases, Upcoming and Top 100.

Find Good Films is a personal project created by Hien Nguyen using HTML, CSS and Javascript. The data on this page belongs to The Movie Database (TMDb).
